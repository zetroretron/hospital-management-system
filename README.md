# How to Run This Project

## Quick Start (3 Steps)

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Initialize database:
```bash
python init_db.py
```

3. Run the application:
```bash
python app.py
```

4. Open browser: http://127.0.0.1:5000
